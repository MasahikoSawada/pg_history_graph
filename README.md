# Usage

``` bash
$ ruby pg_history_graph.rb --help
Usage: pg_history_graph [options]
    -r, --reload                     reload all release note data
    -i, --interval=NUM               header line interval (default: 20)
    -v, --version=NUM                Output releases from NUM version (default: 8)
```
